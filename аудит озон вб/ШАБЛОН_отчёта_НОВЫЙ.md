# КАК РАБОТАТЬ С ПАПКОЙ «аудит озон вб»

## Что здесь есть

| Файл | Для чего |
|---|---|
| `ozon_analyzer.mjs` | Сбор отзывов с Ozon по ссылке |
| `wb_analyzer.mjs` | Сбор отзывов с Wildberries по артикулу |
| `make_pdf.mjs` | Конвертация HTML-отчёта в PDF |
| `ozon_cookies.txt` | Куки для Ozon (обновлять если скрипт перестал работать) |

---

## Процесс аудита — пошагово

**Шаг 1 — Кинуть ссылку в Claude**
Просто отправляю ссылку на товар (Ozon или WB) — Claude сам запускает скрипт и собирает отзывы.

**Шаг 2 — Скинуть скрины карточки**
Скрины всех слайдов инфографики + описание товара + фото первого слайда отдельным файлом в папку.
Claude читает отзывы и скрины и делает анализ.

**Шаг 3 — PDF готов**
Claude создаёт HTML-отчёт и конвертирует в PDF.
Название файла: `Аудит_НазваниеТовара_арт123456.pdf`

---

## Запуск скриптов вручную (если нужно)

```
# Ozon — по ссылке
node ozon_analyzer.mjs https://www.ozon.ru/product/название-123456/

# Ozon — отзывы за период
node ozon_analyzer.mjs https://www.ozon.ru/product/название-123456/ --period=7d
node ozon_analyzer.mjs https://www.ozon.ru/product/название-123456/ --period=30d

# WB — по артикулу
node wb_analyzer.mjs 123456789

# Конвертация HTML в PDF
node make_pdf.mjs имя_файла.html
```

---

## Если куки Ozon протухли

1. Открыть ozon.ru в браузере → войти в аккаунт
2. F12 → вкладка Network → обновить страницу любой товар
3. Найти любой запрос к ozon.ru → Headers → скопировать строку Cookie: ...
4. Заменить содержимое файла ozon_cookies.txt

## Структура готового PDF-отчёта

- Фото товара + ссылка на карточку
- Кто покупает (портрет ЦА)
- Топ-5 причин покупки
- Топ-5 проблем
- Язык покупателей (дословные цитаты)
- Аудит каждого слайда карточки
- Что отсутствует в карточке
- Приоритеты правок (1-2-3)
- Готовый текст описания
- Главный инсайт
